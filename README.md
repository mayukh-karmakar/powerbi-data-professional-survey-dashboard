# Data Professional Survey Breakdown Dashboard

Power BI dashboard project built using a survey dataset to analyze trends in salary, job titles, programming languages, work-life balance, and salary satisfaction among data professionals.

## Files
- `dashboard/data-professional-survey-dashboard.pbix` - Power BI dashboard file
- `data/data-professional-survey-dataset.xlsx` - Source dataset (Excel)
- `images/dashboard-overview.png` - Dashboard preview screenshot

## Dashboard Highlights
- Average salary by job title
- Count of survey takers
- Average age of survey takers
- Work-life balance satisfaction score
- Salary happiness breakdown
- Favourite programming language distribution

## Tools Used
- Power BI
- Power Query
- Excel

## Note
This is a practice/portfolio project for learning data analysis and dashboarding in Power BI.
